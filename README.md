This application offers a full suite of features to manage the entire Lost & Found lifecycle:

Secure User Authentication: Complete system for user Sign-up, Login, and Password Reset (using secure tokens).

Item Reporting: Users can easily report lost or found items with necessary details (type, category, location) via the central API.

Interactive Dashboard: A personalized user dashboard (index.php) displays all active item reports and allows users to track their submissions.

Notification System: Built-in system provides users with alerts regarding potential item matches or status changes on their reports.

Full Item Lifecycle Management: Users can initiate claim requests, and item statuses can be updated to Resolved upon successful return.

Dedicated Admin Panel: Administrators have privileged access (admin.php) to view, manage, and update all reported items, as well as handle user management (role modification, account deletion).

Public Statistics: A landing page (home.php) displays key statistics, such as total users and resolved cases, to showcase the system's effectiveness.
